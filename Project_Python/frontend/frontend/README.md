# F4 Hotel Website

Welcome to the F4 Hotel website project! This project is designed to provide an online platform for users to explore and book rooms at F4 Hotel.

## Project Structure

The project consists of the following files:

- **index.html**: The main page of the hotel website. It includes navigation links to different sections, including a link to the "About Us" page.
- **aboutus.html**: This page presents information about the F4 Hotel team, including backend, frontend, and intern members.
- **room-detail.html**: This file displays detailed information about a specific room in the hotel.
- **app.js**: This JavaScript file handles the loading and rendering of room data from the backend API.
- **css/style.css**: This file contains the CSS styles for the website, ensuring a consistent look and feel across all pages.

## Features

- User-friendly interface for exploring hotel rooms.
- Detailed information about each room.
- Team information available on the "About Us" page.
- Responsive design for optimal viewing on various devices.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the website.
3. Navigate through the pages using the provided links.

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap (for responsive design)
